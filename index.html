How to Install (One-Time Setup)

  1. Download and Extract the ZIP folder.
  2. Run Setup_TubeVault.bat — this starts the engine and prepares Chrome.
  3. In Chrome, ensure Developer mode is ON.
  4. Drag the highlighted 'extension' folder into Chrome.
  5. Once the Welcome Page turns green, you're all set!

✅ Note: You only need to run the setup once. TubeVault will start automatically after that!
