# 🎬 TubeVault - Premium YouTube Video & Audio Downloader

**TubeVault** is a high-performance, premium-designed browser extension that allows you to download YouTube videos in up to **1080p resolution** or extract high-quality **MP3 audio** with just one click.

Designed for creators and collectors who value quality and aesthetics.

---

## ✨ Features

- **Premium Aesthetics**: Modern glassmorphism UI with smooth ambient animations.
- **Smart Detection**: Automatically picks up the YouTube URL from your active tab.
- **High-Quality Formats**: Supports 1080p Full HD, 720p HD, and high-bitrate MP3 extraction.
- **No Fluff**: No ads, no trackers, just high-speed media storage.
- **Direct-to-Device**: Files are saved directly to your local downloads folder.

---

## 🚀 How to Install (One-Time Setup)

Follow these simple steps to add **TubeVault** to your browser:

1.  **Download & Extract**: Get the extension bundle and extract the ZIP folder to your computer.
2.  **Run the Engine**: Double-click **`Setup_TubeVault.bat`**.
    - _This will start the backend engine and automatically prepare your browser._
3.  **Enable Developer Mode**: In the Chrome page that just opened, turn on **Developer mode** toggle (top right corner).
4.  **Add to Chrome**: Drag the highlighted **`extension`** folder directly into the Chrome window.
5.  **Enjoy!**: A "Welcome Page" will confirm you are connected. Pin **TubeVault** to your toolbar and start downloading!

> [!NOTE]
> **One-Time Setup**: Once configured, the TubeVault engine will start automatically every time you log in to your computer!

> [!TIP]
> **Uninstallation**: To remove TubeVault, simply run `Uninstall_TubeVault.bat` and remove the extension from Chrome.
